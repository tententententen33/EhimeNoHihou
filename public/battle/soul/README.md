# ソウル（プレイヤー）画像 — 男の子キャラ・4方向

バトル中のソウル（自分のキャラ）に使う画像です。移動方向で自動的に切り替わります。

## 必要なファイル（透過PNG・ドット絵推奨）

このフォルダ（`public/battle/soul/`）に次の4枚を置いてください。

| ファイル名      | 向き           | 内容                         |
| --------------- | -------------- | ---------------------------- |
| `boy-down.png`  | 下（正面）     | 手前を向いた立ち姿           |
| `boy-up.png`    | 上（背面）     | 後ろを向いた姿               |
| `boy-left.png`  | 左             | 左を向いた横姿               |
| `boy-right.png` | 右             | 右を向いた横姿               |

## 仕様メモ
- 当たり判定は中心の 16×16px のまま。画像は見た目だけ約2.4倍に拡大表示します。
- 各画像は正方形に近い透過PNGがきれいに表示されます（例: 16〜32px四方）。
- どれか欠けている向きは赤ハートにフォールバックします（4枚そろえると常にキャラ表示）。
- 移動方向の主軸（左右 or 上下）で向きが決まり、止まっている間は直前の向きを保ちます。
